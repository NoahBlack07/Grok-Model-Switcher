<h2>Description</h2>
<p>Grok Model Switcher is a Tampermonkey userscript that lets you switch between <strong>Grok-3</strong> and <strong>Grok-2</strong> models on <a href="https://grok.com">grok.com</a>. It patches POST requests to toggle models and adds a header interface to show rate limits and select models.</p>

<p><strong>Key Features:</strong></p>
<ul>
  <li>Toggle between Grok-3 and Grok-2 with one click.</li>
  <li>Show real-time rate limits (remaining/total queries) for both models.</li>
  <li>Patch fetch and XHR requests to use the selected model.</li>
  <li>Use TailwindCSS for a clean UI.</li>
  <li>Update rate limits every 30 seconds.</li>
</ul>

<p>Perfect for testing or using both Grok models while monitoring usage quotas.</p>
